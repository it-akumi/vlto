vlto
====

vlto shows velocity of your projects of [Toggl](https://toggl.com)

## Description

If you use Toggl to track working hours of some projects, this tool can show you velocity of them.

## Requirements

* Go

## Install

```
$ go get -u github.com/it-akumi/vlto
```
